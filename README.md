A small utility to migrate data from [Ditto](http://ditto-cp.sourceforge.net/ "Ditto") SQLite db to [ClipboardFusion](https://www.clipboardfusion.com/) SQLite db.

Migrates text, files (CF_HDROP) & images (CF_DIB).
